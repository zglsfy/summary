  • 3.ls -a 查看当前目录下所有文件（包括隐藏文件）
  • 4.cd path 进入某个目录
  • 5.cd .. 退出当前目录
  • 6.mkdir filename 创建文件夹
  • 7.touch filename 创建文件
  • 8.rm file 移除文件
  • 9.cp file 复制文件
  • 10.mv 移动或者修改文件
  • 11.file/start 查看文件类型或者文件属性信息
  • 12.cat 查看文本文件内容
  • 13.find 查找文件
  • 14.tail -10 查看文件后10行内容
  • 15.head -10 查看文件前10行内容
